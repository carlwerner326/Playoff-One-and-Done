This is an NFL One and Done Fantasy Playoff App
Runs the entire NFL Playoffs
Scored with the same metrics as DraftKings
Has a live leaderboard that runs total points throughout


Features it has:
  Login / PIN
  Roster rules
  Burn / lock behavior
  FLEX + QB limit

Tech stack:
  Streamlit
  Python
  JSON storage
  Deployed on Streamlit Cloud

Status: 
Active development
Built for private league use among friends
